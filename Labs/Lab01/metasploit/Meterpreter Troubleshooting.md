# Meterpreter Troubleshooting

## Kill all jobs (handlers)

List jobs with `jobs` command

Kill all the listed jobs with `jobs -k <Id>` (substitute <Id> with the number from the jobs command)

![image.png](Meterpreter%20Troubleshooting/image.png)

## Kill all sessions (established connections)

List sessions with `sessions` command

Kill all the listed sessions one at a time with `sessions -k <Id>`(substitute <Id> with the number from the sessions command)

![image.png](Meterpreter%20Troubleshooting/image%201.png)
